### Frontend Service
- **Root Directory**: `/frontend`
- **Framework**: Create React App
- **Build Command**: `yarn install && yarn build`
- **Start Command**: `npx serve -s build -l ${PORT}`
- **Output Directory**: `build`
- **Environment Variables**:
  ```
  REACT_APP_BACKEND_URL=https://your-backend-service-url.zeabur.app
  ```
  (Replace with actual backend URL after backend deploys)

## Deployment Steps

1. **Deploy Backend First**:
   - Create new service in Zeabur
   - Connect to your GitHub repo
   - Set root directory to `/backend`
   - Add all backend environment variables
   - Deploy and get the backend URL

2. **Deploy Frontend Second**:
   - Create new service in Zeabur
   - Connect to same GitHub repo
   - Set root directory to `/frontend`
   - Add `REACT_APP_BACKEND_URL` with backend URL from step 1
   - Deploy

3. **Update CORS in Backend** (if needed):
   - Go to backend service settings
   - Ensure `ALLOWED_ORIGINS` includes your frontend URL

4. **Link Custom Domain**:
   - In Zeabur, go to frontend service settings
   - Add domain: `performos.digital`
   - Follow DNS configuration instructions

## Seeding Database

After first deployment, seed the database:
```bash
curl -X POST https://your-backend-url.zeabur.app/api/seed
```

Or run seed script manually via Zeabur console.
